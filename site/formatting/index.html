    Text formatting¶

    Text formatting allows you to create powerful messages for different users and context and increase your audience range. It will replace some prefixed text with some dynamic text, for instance in Welcome service you can set a welcome message with text formatting to change the text based on the user who joined. e.g. Welcome {user} to {guild} with {memberCount} user/s Now the {user}, {guild}, {memberCount} variables will be replaced at runtime with the appropriate data. e.g. output Welcome Waterball to April Testing with 3 user/s.

    You can use text formatting in any service that allow you to set custom message, you can use choose from the above tags and use them in the message. e.g. Welcome {user} in {guild}!! You're the {memberCount}th member. Ours server has been founded {guild.created.ago} and continuously increasing! All tags use curly brackets between the key so always make sure to use them, and they'll be replaced at runtime with the appropriate value.

    Danger

    This is a new feature is may not be available in every service and keep in mind that not every formatting are available to use some of them are only available for some type of service, for example {before.user} is not available on reaction role but only on logs

    User Joined¶

    Include all tags from guild and user, no extra tags are available

    User Left¶

    Include all tags from guild and user, no extra tags are available

    User update username¶

    Include all tags from guild, user, before user and after user

    User update avatar¶

    Include all tags from guild, user, before user and after user

    User update discriminator¶

    Include all tags from guild, user, before user and after user

    Message delete¶

    Include all tags from guild, user, channel and include {message} extra tag that contains the content of the message

    Message update¶

    Include all tags from guild, user, channel and include {before.message} that contains the previous content of the message and {after.message} that contains the modified content.

    Voice moved¶

    Include all tags from guild, user, channel, before channel and after channel

    Voice Joined¶

    Include all tags from guild, user, channel, before channel and after channel

    Voice Left¶

    Include all tags from guild, user, channel, before channel and after channel

    User update nickname¶

    Include all tags from guild, user, before user and after user

    User remove nickname¶

    Include all tags from guild, user, before user and after user

    User add role¶

    Include all tags from guild, user, before user and after user, include {roles} that contains the added roles

    User add role¶

    Include all tags from guild, user, before user and after user, include {roles} that contains the removed roles

    Text channel created¶

    Include all tags from guild and channel

    Text channel destroyed¶

    Include all tags from guild and channel

    User banned¶

    Include all tags from guild, user

    User unbanned¶

    Include all tags from guild, user

    Voice channel created¶

    Include all tags from guild and channel

    Voice channel destroyed¶

    Include all tags from guild and channel

    Channel update¶

    Include all tags from guild, channel, before channel and after channel
